# Planner Bot - Telegram бот для планирования

Бот для создания событий и экспорта в календарь (Google Calendar, Apple Calendar).

## Установка

1. Клонируйте репозиторий или скопируйте файлы проекта

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Получите токен бота в @BotFather и добавьте его в `.env`:
```
TELEGRAM_BOT_TOKEN=your_token_here
```

## Запуск

```bash
python bot.py
```

## Команды

- `/start` - Приветствие и информация о боте
- `/plan` - Создать новое событие
- `/my_events` - Показать все мои события
- `/delete <id>` - Удалить событие по ID
- `/help` - Справка

## Как это работает

1. Пользователь отправляет команду `/plan`
2. Бот запрашивает название, дату, время и описание события
3. После создания бот отправляет `.ics` файл
4. Пользователь открывает файл и добавляет событие в календарь

## Структура проекта

```
├── bot.py              # Основная логика бота
├── database.py         # Работа с SQLite базой данных
├── calendar_export.py  # Генерация .ics файлов
├── config.py           # Конфигурация
├── requirements.txt    # Зависимости
└── .env.example        # Пример переменных окружения
```
